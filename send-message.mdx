---
title: "카나페로 데이터 전송하기"
---

<Info>
  동적으로 값을 변경해야 하는 등의 특별한 경우가 아닌 이상 초기화(Initializing) 또는 파라미터에서 미리 모든 값을
  넣어두는 것을 권장합니다. [연동 시작하기](/quickstart)
</Info>

## 개별 데이터 전송

- customUserId

```typescript
CanapeEmbedHelper.event.send(type: "customUserId", data: string): void
```

- isParticipationBlocked

```typescript
CanapeEmbedHelper.event.send(type: "isParticipationBlocked", data: boolean): void
```

- signupSession

```typescript
CanapeEmbedHelper.event.send(type: "signupSession", data: string): void
```

- isForceNewSignupUser

```typescript
CanapeEmbedHelper.event.send(type: "isForceNewSignupUser", data: boolean): void
```

## 모든 데이터 일괄 전송

```typescript
CanapeEmbedHelper.event.set(data: {
    customUserId?: string
    isParticipationBlocked?: boolean
    signupSession?: string
    isForceNewSignupUser?: string
}) => void
```
